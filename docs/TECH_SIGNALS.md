# Technical Signals

We aggregate 4 factor groups (25% each):
- **Trend**
- **Momentum**
- **Volatility**
- **Flow**

Roll-ups:
- `TA_Score` ∈ [0, 100]
- `Macro_Adj` ∈ [−10, +10]
- `Final_Signal` ∈ {BUY, HOLD, SELL}

Inputs commonly include SMA/EMA slopes, RSI14, MACD, ATR%, Volume/Flow metrics, etc.
